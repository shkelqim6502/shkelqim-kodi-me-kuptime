# shkelqim-kodi-me-kuptime
 Kodi pa kuptim është si drita pa ngjyrë – ndriçon, por nuk frymëzon.         Kur kuptimi udhëheq kodin, edhe gabimi bëhet mësim.
Shtova një rresht definimi të kuptimit të kodit
Kodi është një grup udhëzimesh të shkruara në një gjuhë programuese që i thotë kompjuterit çfarë të bëjë. Ai mund të përdoret për të krijuar aplikacione, faqe interneti, lojëra, sisteme operative dhe shumë më tepër.
